# Repository Notes

This repository is prepared as a clean source project for GitHub publication.

Included:

- Unity source assets required to open and run the project
- Scene generator, scripts, tests, project settings, and package manifests
- Local material and texture assets used by the current playable demo

Excluded:

- `Library`, `Logs`, `Builds`, `UserSettings`, and other generated Unity artifacts
- local screenshots, test output XML, and temporary debug files
- exploratory review images that are not yet integrated into the shipped project

If integrated art or polished screenshots are added later, they should be renamed and moved into a stable tracked folder before commit.
